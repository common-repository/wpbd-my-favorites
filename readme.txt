== Description ==
Wpbp My Favorites allow you to display favorites post by thumbnail (compatible to Myarcadeplugin and wordpress thumbnail) or link on buddypress profil page.
(Buddypress and WP Favorite Posts must be installed)

== Installation ==

1. Open `wp-content/plugins` Folder
2. Put: `Folder: wpbd-my-favorites`
3. Activate `Wpbp My Favorites` Plugin
4. Go to `WP-Admin -> Wpbp My Favorites`

== Frequently Asked Questions ==


= How To display favorites =
1. Go to `WP-Admin -> Options -> Wpbp-MyFavorites`.
2. Type any title you like in the name link.
3. Type the number of favorites you want display and how, select Thumbnail or link.
